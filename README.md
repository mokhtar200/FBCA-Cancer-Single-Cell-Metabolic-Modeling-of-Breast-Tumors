# FBCA-Cancer-Single-Cell-Metabolic-Modeling-of-Breast-Tumors

Flux Balance Cellular Automata framework integrated with
single-cell RNA-seq to study metabolic heterogeneity and
drug-resistant cell states in breast cancer.

## Methods
- scRNA-seq (Seurat)
- Single-cell metabolic scoring (GSVA)
- Flux-informed Cellular Automata simulation

## Applications
- Cancer metabolism
- Drug resistance
- Systems biology
- Precision oncology
